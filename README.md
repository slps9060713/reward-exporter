# 🎲 Twitch 獎勵抽獎系統

一個簡單易用的 Twitch 頻道點數獎勵抽獎工具，支援輪盤抽獎和縮圈功能。

## ✨ 功能特色

- 🎯 **公平抽獎** - 機率均等的輪盤抽獎系統
- 🔢 **縮圈功能** - 逐步公開百位、十位、個位數字
- 📋 **多獎勵支援** - 為每個獎勵創建獨立分頁
- 🎨 **自定義模式** - 無需登入，支援自定義號碼範圍
- 🔐 **Twitch 整合** - 直接載入頻道點數獎勵兌換名單
- 📱 **響應式設計** - 支援電腦和手機

## 🚀 快速開始

### 部署到 GitHub Pages

1. **上傳到 GitHub**
   ```bash
   git add .
   git commit -m "Deploy lottery system"
   git push origin main
   ```

2. **啟用 GitHub Pages**
   - 前往 GitHub 專案的 **Settings** → **Pages**
   - 選擇 **Branch**: `main`，**Folder**: `/docs`
   - 點擊 **Save**

3. **訪問您的網站**
   - 網址：`https://您的用戶名.github.io/專案名稱/`
   - 大約 1-5 分鐘後網站就會上線

### 設定 Twitch 應用程式（選擇性）

如果要使用 Twitch 登入功能：

1. 前往 [Twitch 開發者控制台](https://dev.twitch.tv/console)
2. 點擊「註冊應用程式」
3. 填寫資訊：
   - **名稱**：Twitch 獎勵抽獎系統
   - **OAuth 重定向 URL**：`https://您的用戶名.github.io/專案名稱/`
   - **類別**：應用程式整合
4. 記下您的 **Client ID**
5. 編輯 `docs/app.js`，將第 2 行改為您的 Client ID：
   ```javascript
   const APP_CLIENT_ID = '您的ClientID';
   ```
6. 重新推送到 GitHub

## 📖 使用說明

### 使用 Twitch 獎勵抽獎

1. 點擊「連接 Twitch 帳號」登入
2. 系統自動載入所有獎勵作為分頁
3. 切換到要抽獎的獎項
4. 查看左側參與者列表（001, 002, 003...）
5. 點擊「啟動抽獎」開始
6. 輪盤會快速旋轉並選出號碼
7. 點擊「縮圈」逐步公開數字
   - 第一次：公開百位數
   - 第二次：公開十位數
   - 第三次：公開個位數
8. 不符合的號碼會被淘汰（變灰刪除線）
9. 中獎號碼會以綠色高亮顯示

### 使用自定義模式

1. 點擊「跳過登入，使用自定義模式」
2. 輸入最大號碼（例如：100）
3. 點擊「更新列表」
4. 系統生成 001 到該號碼的列表
5. 點擊「啟動抽獎」
6. 使用縮圈功能逐步揭曉中獎號碼

## 🐛 常見問題

### Q: 為什麼會出現 redirect_mismatch 錯誤？
A: 這是因為使用了預設的 Client ID，請按照上方「設定 Twitch 應用程式」步驟創建自己的應用程式。

### Q: 可以不登入 Twitch 使用嗎？
A: 可以！點擊「跳過登入，使用自定義模式」即可使用自定義號碼抽獎。

### Q: 縮圈功能是什麼？
A: 縮圈會逐步公開中獎號碼的每個位數，同時淘汰不符合的參與者，增加抽獎的緊張感和趣味性。

### Q: 抽獎是否公平？
A: 是的！使用 JavaScript 的 Math.random() 函數，每個參與者的中獎機率完全相同。

### Q: 可以重複抽獎嗎？
A: 可以！每次抽獎都是獨立的，可以重複進行。
